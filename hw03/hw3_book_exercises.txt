Q1:What possible values can a Boolean expression have?
A:There are two boolean values.They are True and False.

Q2:Where does the term Boolean originate?
A:The term Boolean comes from the name of the British mathematician George Boole.

Q3:What is an integer equivalent to True in Python?
A:Python treats every integer value(positive and negative) other than 0, to be true.

Q4:What is the integer equivalent to False in Python?
A:Python considers the integer value zero to be false.

Q5:Is the value -16 interpreted as True or False?
A:True.
